# Softweare Engineering Project T1 Jan 2025

## Project Overview
Softweare Engineering Project T1 Jan 2025 is a cutting-edge software development project designed to showcase best practices in modern web development. This project leverages Node.js and npm to build a robust, scalable, and efficient application.

## Prerequisites
Before running the project, ensure you have the following installed on your system:

- **Node.js**: v20.14.0 ([Download Node.js](https://nodejs.org/en/download/))
- **npm**: v10.8.1 (included with Node.js)

## Installation & Setup
Follow these steps to set up and run the project:

### 1. download the folder and unzip and head over to the folder you have downloaded, the project strucure in the folder will look like below
## Project Structure
```
│── src/              # Source code
│── public/           # Static assets
│── package.json      # Project dependencies
│── README.md         # Documentation
│── .gitignore        # Git ignored files
```

### 2. Install Dependencies
Open the terminal in the project directory and run:
```sh
    npm install
```

### 3. Run the Development Server
To start the development server, execute:
```sh
    npm run dev
```

### 4. Build the Project
For production build, run:
```sh
    npm run build
```
